Group project of 4. 

Application where users can offer items they no longer need to others (For Free). 

We included many features, like:
- Sign up 
- Login 
- Reset password
- Email confirmation
- Edit user profile
- Post ads 
- Update ads
- Delete ads
- Contact other users 
- Search ads by keywords 
- Filter by categories and more.

Completed: October 2021
